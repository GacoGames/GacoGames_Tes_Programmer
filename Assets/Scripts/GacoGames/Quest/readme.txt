Main Quest
- Sequential quest chains to progress the main story
- Auto assigned by system

Side Quest
- Quest received from NPCs or certain event triggers
- Isn't shown in Journal/Quest menu until received
- Example : Talk to a troubled farmer, and accept his quest to protect the farm -> Quest assigned

Special Quest
- Quest received from Game Lobby, usually for character introduction
- Isn't shown in Journal/Quest menu until received
- Example : New Character Banner -> Start Hero Quest -> Quest assigned

Event Quest
- Quest received from Game Lobby, for time-limited events
- Isn't shown in Journal/Quest menu until received
- Example : Summer Event -> Start -> Quest assigned
- Event Quest has EndDate (usually same with the running event) and will be removed on time-out
